Contact Information

Reporting an Issue

We use the GitHub Issues Tracker to keep track of requested changes in the project.

If you would like to report a correction, missing meet, feature suggestion, or basically anything else, the absolute best place to do that is in the issue tracker linked above. Creating an account is extremely easy, and you can optionally receive e-mail notifications when we finally make the changes.

For corrections to data, please include a link to the corrections, preferably from a federation-linked source. Video evidence is also acceptable in the case of federation error or omission.

If a meet is missing, and you would like it to be added, please feel free to file an issue. If the meet is a recent meet in the USPA, RPS, or USAPL, there is a good chance it will be added within a week. Other federations are updated on a less frequent basis, as they require manual checking.

E-Mail

Please address all other inquiries to: updates@openpowerlifting.org.

If you would like to contribute to the project, please send an email for an invite to the team chat. Help is always appreciated. Every meet requires manual work, and there's more than enough to go around.
